Generate a public and private key pair with a modulus of size bits.

Initialise a key pair with modulus size bits. It is required that pk and vk are initialised before calling this function. pk and vk are also expected to not be NULL.

1 pcs2_public_key *pk = pcs2_init_public_key();
2 pcs2_private_key *vk = pcs2_init_private_key();
3 pcs2_generate_key(pk, vk, 2048);

In practice the bits value should usually be greater than 2048 to ensure sufficient security.

Parameters
pkAn initialised pointer to a pcs2_public_key.
vkAn initialised pointer to a pcs2_private_key.
bitsThe required number of bits the modulus n will be.

int pcs2_verify_private_key ( pcs2_private_key *  vk)

Verify a private keys values.

This can be used as a sanity check to ensure that the pcs2_private_key is valid. Note, that this only checks what it can, and using it with a non-matching private key can still produce unintended results. This will primarily be of use when importing a private key from a file.

1 pcs2_private_key *vk = pcs2_init_private_key();
2 pcs2_import_private_key(vk, "key.private");
3 if (!pcs2_verify_public(vk)) exit(0);
Parameters
vkA pointer to an initialised pcs2_private_key.
Returns
non-zero on valid key, 0 on invalid key.

void pcs2_clear_public_key ( pcs2_public_key *  pk)

Zero all fields in a pcs2_public_key.

This function zeros all data in pk. It is useful to use if we wish to generate or import a new key value and want to safely ensure we have removed the last keys values.

1 // ... Initialised a key pk, and done some work with it
2 
3 pcs2_clear_public_key(pk); // All data from old key is now gone
4 pcs2_import_public_key(pk, "public.key"); // Safe to reuse this key, still initialised
